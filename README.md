### 赛题理解
本次比赛关于一个四分类问题，其中标签为onehot格式即[0,1,0,0]的格式，id为唯一标签，因此理论是不存在意义，但是后续可以
简单探究下id值是否与标签相关。其中唯一变量为一串由逗号分隔的心跳信号，已脱敏为数值。
评价指标：赛题中已给出为sum(sum(abs(pre-true))),即理解为各种类别预测概率与真实概率的差值的绝对值之和
